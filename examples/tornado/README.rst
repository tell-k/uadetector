====================
Example for Tornado
====================

Setup
======

::

 $ pip install uadetector
 $ pip install tornado

Run
===========

::

 $ python main.py

You can access http://127.0.0.1:8888. and get output like bellow.

::

 {
   "device_type": "pc",
   "os": "Mac OSX",
   "browser": "Chrome",
   "from_pc": true,
   "from_smartphone": true
 }
